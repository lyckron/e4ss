Working with the init object

Default (display block, align left, light theme, platform colors)

            
                $('div.e4ss-wrap').e4ss();

Align center

            
                $('div.e4ss-wrap').e4ss(
                    init: {
                    align: 'center'
                    }
                );

Align right

            
                $('div.e4ss-wrap').e4ss(
                    init: {
                        align: 'right'
                    }
                );

Default inline ex

            
                $('div.e4ss-wrap').e4ss(
                    init: {
                        display: 'inline'
                    }
                );

Inline align center

            
                $('div.e4ss-wrap').e4ss(
                    init: {
                        display: 'inline',
                        align: 'center'
                    }
                );

Inline align right

            
                $('div.e4ss-wrap').e4ss(
                    init: {
                        display: 'inline',
                        align: 'right'
                    }
                );

Default dark theme - with default colors, and custom text shadow

            
                $('div.e4ss-wrap').e4ss(
                    init: {
                        textShadow: '1px 1px #000000',
                        backgroundColor: 'dark'
                    }
                );

Inline dark theme - with custom color and text shadow

            
                $('div.e4ss-wrap').e4ss(
                    init: {
                        display: 'inline',
                        align: 'center',
                        color: '#d1d1d1',
                        textShadow: '1px 1px #000000',
                        backgroundColor: 'dark'
                    }
                );

Custom Wrap Padding and Font Size

            
                $('div.e4ss-wrap').e4ss(
                    init: {
                        wrapPadding: '20px',
                        fontSize: '1.5em'
                    }
                );

Custom Border and Background Color

            
                $(div.e4ss-wrap).e4ss(
                    init: {
                        border: '2px solid #ADD8E6',
                        backgroundColor: 'white'
                    }
                );

Custom text for all platforms

            
                $(div.e4ss-wrap).e4ss(
                    init: {
                        text: 'Please, Share!'
                    }
                );

Use Share Icon for all platforms

            
                $(div.e4ss-wrap).e4ss(
                    init: {
                        text: '#icon'
                    }
                );

Working with Platform Objects

Don't include a platform, ex. Twitter

            
                $('div.e4ss-wrap').e4ss({
                    twitter: {
                        include: false
                    }
                });

Custom Flat UI Style

            
                $(div.e4ss-wrap).e4ss(
                    init: {
                        color: '#ffffff',
                        border: 'none',
                        backgroundColor: ''
                    },
                    facebook: {
                        backgroundColor: '#3b5998'
                    },
                    twitter: {
                        backgroundColor: '#00aced'
                    },
                    linkedin: {
                        backgroundColor: '#007bb6'
                    },
                    google: {
                        backgroundColor: '#d34836'
                    }
                );

Custom platform color and text

            
                $('div.e4ss-wrap').e4ss({
                    facebook: {
                        color: 'green',
                        text: 'Share on Facebook'
                    },
                    twitter: {
                        color: 'orange',
                        text: 'Share on Twitter'
                    },
                    linkedin: {
                        color: 'red',
                        text: 'Share on LinkedIn'
                    },
                    google: {
                        color: 'blue',
                        text: 'Share on Googleplus'
                    }
                });

Mix share icons and text - if you for some reason would do that

            
                $('div.e4ss-mix-icon-and-text').e4ss({
                    init: {
                        text: '#icon'
                    },
                    twitter: {
                        text: 'Share on Twitter'
                    },
                    google: {
                        text: 'Share on Googleplus'
                    }
                });
